Статус сервиса

Production‑контур активен: публичный сайт работает в корне домена, web‑клиент открыт отдельно по /web/, а скачивание клиентов вынесено в downloads manifest.

Клиенты и версии

  • Web/PWA: основной browser‑вход, manifest scope — /web/.
  • Android: debug APK для ручной проверки, не production release.
  • macOS: ZIP для проверки desktop‑клиента, перед публичным релизом нужна подпись.
  • CLI: архив клиента и update HTTPS доступны через production update contour.

Проверка целостности

Контрольные суммы опубликованы в checksums.txt. Если приложение не подключается, проверьте сеть и попробуйте открыть web‑клиент заново.
