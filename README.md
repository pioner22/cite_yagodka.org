# cite_yagodka.org

Репозиторий статического сайта `yagodka.org`.

## Структура

- `index.html` — входная страница сайта.
- `assets/` — общие ассеты (картинки/иконки/стили).
- `pages/` — статические страницы (если есть).
- `web/` — собранный web/PWA клиент Yagodka (публикуемая сборка).

## Как обновлять web/PWA

Сборка web‑клиента лежит в основном монорепозитории `yagodka`:

1) Собрать web‑клиент (в `yagodka/client-web`).
2) Скопировать результат сборки в `www/web/`.
3) Обновить этот репозиторий (`web/`), сделать commit и push.

## Безопасность

- Не добавляйте в репозиторий ключи, пароли, `service.env` и любые приватные конфиги.
- Не храните персональные данные пользователей/ID в статических файлах.
